Flutter and Dart brand development

Building developer brand kits as a service

We worked closely with the Flutter and Dart teams to create brand visuals and a series of websites that clearly communicate the unique value of each property through visually rich and memorable web experiences. Beyond engaging and kinetic homepages, both sites required the ability to house an extensive amount of reference material, documentation, and code examples. We knew developers would continually reference the documentation pages, and so made them easy to read and navigate.

Visualizing the language

Dart is the programming language behind the Flutter SDK. Flutter developers appreciate Dart for its features— Hot Reload—which allow for quick and seamless deployment across a range of platforms.

Designing and deploying beautiful apps

Flutter impressed users with its ability to create beautifully crafted, natively compiled applications for mobile, web, and desktop from a single codebase. We showcased those capabilities on the Flutter website through a series of fluid, code-driven animations that visualized real-life examples of beautiful sites built with the Flutter SDK.

Creating connections

We worked closely with stakeholders across the Flutter and Dart teams to build a visual strategy that communicated the strong connection between both properties. The Dart brand palette builds from the same visual language as Flutter, but as a dark-themed counterpart. Darker shades reference coding terminals, which allows code snippets to emerge as primary design elements.

Hitting the ground running

Developers familiar with Java, Swift, and JavaScript quickly learn Dart.. We wanted to showcase the ease with which they learn Dart by allowing users to directly try the language from both homepages.

We also helped redesign DartPad, a lightweight coding environment that works as a standalone site and an embeddable element. DartPad incorporates UI elements from Material Design, and supports dark and light themes.
