Learning Pathways

We excel at building sequential pathways of codelabs, devsite articles, and videos that teach developers how to build anything with Google’s technologies, then reward them with a badge upon completion. Whether online or in person, pathways provide developers with a logically ordered way to learn a single, clear lesson that they’ve always wanted to understand, but never knew where to start. For example, we designed the pathway about how to host web apps on Google Cloud for developers who understand the basics of app development, but possess minimal Google Cloud experience.

Topic Pages

Sometimes, developers don’t need to learn something new, but would like to stay on top of the latest tips, tricks, and versions released. This is why we created topic pages, which host static and dynamic content, including blog posts, experts and influencers, news articles, and videos. For example, the Cloud CI/CD topic page blends static and dynamic content that gives developers the most updated information about CI/CD.

Quizzes and Badges

We work with you to build quizzes that developers need to complete at the end of pathways, which allows you to ensure that your learning objectives are met. In return for completing quizzes, we reward developers with badges that they can add to their profiles.
